# 🌌 Project Vision

> **"Transforming raw financial transaction data into an intelligent fraud detection system."**

This project is an **end-to-end Machine Learning application** designed to identify whether a credit card transaction is **fraudulent or legitimate** using transaction-based data patterns.

It combines:

- 🧠 **Machine Learning intelligence**
- ⚡ **FastAPI backend deployment**
- 🌐 **Interactive frontend interface**
- 📊 **Real-time prediction flow**

---

# 🎯 Problem Statement

Credit card fraud is one of the biggest threats in the modern digital payment ecosystem.  

### ❗ The challenge:
- Fraud is **rare**
- Hard to detect
- Hidden among legitimate transactions

### ✅ The solution:
Using **Machine Learning classification models** to detect fraud patterns in real-time.
